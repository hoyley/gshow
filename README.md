# gshow

Prerequisites:
- Java9 JDK or higher
- Gradle
- NPM

Get the code:
```
git clone https://github.com/hoyley/gshow.git
cd gshow
```

# Local Setup

Build the server:
``` 
gradle build
```

Run the server:
```
gradle bootRun
```

Build the client:
```
cd app
npm install
```

Run the client:
```
npm start
```

Navigate to `localhost:3000` to open the client in the browser.
